byte | 1 byte
short | 2 bytes
int | 4 bytes
long | 8 bytes

float | 4 bytes
double | 8 bytes

char | somente um caracter
boolean | false or true


Conversão unidades medida

fator menor para fator maior = OK, tranquilo, pois tudo que é menor irá caber no fator maior.
fator maior para fator menor = pode gerar problemas, pois pode ser que não aconteça a conversão devido ao tamanho do arquivo.
